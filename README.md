# Game Library 

This project is a game library application .

The goal of this project was to practice working with JavaScript objects, constructors, and DOM manipulation while building a more interactive application.

## Live Demo

https://alperen-isler.github.io/game-library/

## Built with

- HTML
- CSS
- JavaScript

## ✨ Features

- Add new games to the library
- Display games as cards
- Store game information such as title, hours played, platform and completion status
- Add button to toggle game completion status
- Add button to remove games from the library
- Dynamic updates without refreshing the page
